# 🚀 业务项目

按项目组织的决策、实现、踩坑、架构演进记录。

## 分类

- [集群管理控制台](/topics/projects/cluster-console/) - React 前端项目，K8s 控制台（GPU 监控、云监控、容灾演练、节点监控等）
- [DRMS 容灾系统](/topics/projects/drms/) - Next.js 项目，容灾演练任务引擎

## 与技术分类的区别

- **技术分类**（`frontend/`、`k8s/` 等）：放**可复用**的通用知识
- **项目分类**（本目录）：放**项目专属**的决策、模块实现、业务约定

同一知识点可以在两处都出现——技术分类写抽象要点，项目分类写具体落地并链接过去。
