## 要点回顾

### 云原生解决的问题
1. 环境不一致 → 容器化（Docker）
2. 手动扩缩容 → 弹性编排（K8s）
3. 单体变更慢 → 微服务
4. 发布周期长 → CI/CD
5. 运维复杂 → 可观测性（Prometheus + Grafana + Jaeger）
6. 基础设施耦合 → 声明式 + 不可变基础设施

### 关键里程碑
- Docker (2013) — 容器民主化
- Kubernetes (2014) — 编排标准统一
- CNCF (2015) — 生态治理
- Istio (2017) — 服务网格
- eBPF (2021) — 内核可编程
- AI+Cloud Native (2024) — GPU 成为一等公民

### 与工作关联的 CNCF 项目
- NVIDIA Device Plugin / GPU Operator — qGPU 调度底层
- Volcano — 腾讯开源的批调度器（CNCF 孵化）
- Prometheus + Grafana — GPU 监控体系
